# XSD Explorer

## Overview
A Next.js web application for browsing and exploring XSD (XML Schema Definition) type definitions. It provides an interactive catalog of XSD objects with search, filtering, and detailed type views.

## Project Architecture
- **Framework**: Next.js 15.5.10 with React 19, TypeScript
- **Build**: Static export (`output: 'export'` in next.config.mjs)
- **Data**: Pre-built JSON index (`public/data/xsd-index.json`) generated by Python script
- **Deployment**: Static site (builds to `out/` directory)

## Key Files
- `app/` - Next.js app directory (layout, page, components)
- `app/components/explorer-app.tsx` - Main explorer component
- `lib/xsd-types.ts` - TypeScript type definitions for XSD data
- `scripts/build_xsd_index.py` - Python script to generate XSD index data
- `public/data/xsd-index.json` - Pre-built XSD data index
- `next.config.mjs` - Next.js configuration

## Development
- Dev server: `npm run dev` (runs on port 5000, host 0.0.0.0)
- Build: `npm run build` (static export to `out/`)

## Deployment
- Target: Static site
- Public directory: `out/`
